# Join Gate Bot

Простой Telegram-бот для групп и супергрупп с топиками.

## Что делает

- приветствует новых участников;
- мутит их до тех пор, пока они вручную не добавят нужное число участников;
- удаляет сообщения старых участников, пока те не выполнят условие;
- работает в форумах и умеет писать в заданный топик;
- хранит настройки в SQLite;
- тексты лежат отдельно в `texts.json`.

## Запуск

1. Установи зависимости:
   ```bash
   pip install -r requirements.txt
   ```

2. Задай токен:
   ```bash
   export BOT_TOKEN="123456:ABC..."
   ```

3. Запусти:
   ```bash
   python main.py
   ```

## Команды для админов чата

- `/setrequired N`
- `/settopic`
- `/settopic off`
- `/settopic 123`
- `/settext KEY новый текст`
- `/gettexts`
- `/reloadtexts`
- `/config`

## Важно

Бот должен быть администратором группы с правом ограничивать участников. Для форумов Telegram использует `message_thread_id` у сообщений в топиках, а `restrictChatMember` требует админские права.
